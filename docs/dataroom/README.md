## Strategy: prime focus + 0.5%-on-every-deal

Target prime areas (Atlantic Seaboard, City Bowl, Southern Suburbs, Constantia; then prime
JHB-north and Umhlanga) where **R6m+ values are normal**, with a lean concierge team. The free
0% tier requires a **qualifying bond (≥80% LTV)** so we earn the bank-paid origination referral;
**cash / sub-80% deals pay a 0.5% facilitation fee** — so **every deal monetises at ~0.5% of
value** (still ~10–14× cheaper than agent commission). This closes the cash-deal gap.

## Headlines (base case)

- **Revenue:** R1.8m (Y1) → **R80.0m (Y5)**; **EBITDA-positive in Y3**, ~50% margin by Y5.
- **Model:** ~0.5% of value per deal (**~R45–50k**) from the bond referral *or* the facilitation
  fee, plus panel conveyancing and premium add-ons — vs the **R345k–R483k** an agent would charge.
- **Capital:** **~R11m total to breakeven** — comfortably fundable on a **single ~R15m seed**
  (growth capital, not survival capital).
- **Valuation:** seed pre-money **~R50–60m**; Year-5 enterprise value **~R320–480m** (4–6×
  revenue), with the ~50% margin supporting the upper half.

## Before circulating

These documents are **model-grade, not audit-grade**. Re-confirm the four highest-leverage
inputs against primary sources first — the **0.5% bond referral share**, the **conveyancing
panel %**, **Western Cape/CT transfer counts**, and **SA CAC** (see `00-assumptions.md §F`).
None changes the model's shape; they sharpen individual lines.
